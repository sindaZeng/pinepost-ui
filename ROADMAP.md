# Pinepost UI Roadmap

Pinepost UI is moving toward a complete, commercial-friendly React component library with a warm forest-post visual language.

## Near Term

- Harden current v0.10 components with more keyboard, focus, and edge-case tests.
- Expand the docs with per-component disabled states, edge cases, and copyable recipes.
- Publish the demo to GitHub Pages from `main`.

## Mid Term

- Add saved table view persistence examples for app-owned settings.
- Add richer formatting presets for date and time panels.
- Add Cascader multi-select exploration and deeper keyboard coverage.
- Add more theme token customization examples.
- Add visual regression checks for the demo catalog.

## Release Policy

- Patch releases fix bugs, type issues, and documentation mistakes.
- Minor releases add components or public API.
- Major releases are reserved for breaking API or token changes.
