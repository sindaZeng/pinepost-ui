# Pinepost UI Roadmap

Pinepost UI is moving toward a complete, commercial-friendly React component library with a warm forest-post visual language.

## Near Term

- Keep the v0.27 Recipe Gallery team handoff flow as the check for saved bundle import, apply, recovery, and metadata preview.
- Keep the v0.25 Commercial Pressure Lab stable as the heavy Table, Form, and Upload regression baseline.
- Keep display-oriented components queued until saved workflow handoff and heavy workflow maturity stay strong.

## Mid Term

- Add deeper saved bundle handoff recipes only after team-owned commerce and learning handoff previews remain stable.
- Continue package release checklist automation around GitHub release creation and npm verification.
- Add more copyable recipes after bundle import and recovery behavior remains stable.

## Release Policy

- Patch releases fix bugs, type issues, and documentation mistakes.
- Minor releases add components or public API.
- Major releases are reserved for breaking API or token changes.
