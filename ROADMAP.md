# Pinepost UI Roadmap

Pinepost UI is moving toward a complete, commercial-friendly React component library with a warm forest-post visual language.

## Near Term

- Harden current v0.2 components with more keyboard, focus, and edge-case tests.
- Add form-oriented components: InputNumber, Rate, FormField, and ValidationMessage.
- Expand the docs with more edge-case examples and copyable recipes.
- Publish the demo to GitHub Pages from `main`.

## Mid Term

- Add data components: Table, Tree, Calendar, Carousel, Image, Transfer, and Cascader.
- Add feedback utilities: Loading, Message, Notification, Tour, and Affix.
- Add theme token documentation and customization examples.
- Add visual regression checks for the demo catalog.

## Release Policy

- Patch releases fix bugs, type issues, and documentation mistakes.
- Minor releases add components or public API.
- Major releases are reserved for breaking API or token changes.
