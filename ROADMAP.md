# Pinepost UI Roadmap

Pinepost UI is moving toward a complete, commercial-friendly React component library with a warm forest-post visual language.

## Near Term

- Keep the v0.25 Commercial Pressure Lab as the handoff check for server Table, dynamic Form, and controlled Upload workflows.
- Keep display-oriented components queued until heavy workflow maturity is stronger.
- Deepen saved bundle handoff examples after the Table/Form/Upload workflow pass stays stable.

## Mid Term

- Add saved bundle handoff examples for commerce launches and learning flows.
- Continue package release checklist automation around GitHub release creation and npm verification.
- Add more copyable recipes for commerce launches and learning-app flows.

## Release Policy

- Patch releases fix bugs, type issues, and documentation mistakes.
- Minor releases add components or public API.
- Major releases are reserved for breaking API or token changes.
